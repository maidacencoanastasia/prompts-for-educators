# Prompt Library for Educators — USM

A comprehensive, multilingual AI prompt library designed to empower educators with ready-to-use prompts for various teaching scenarios and subject areas.

## Overview

This prompt library provides educators with curated AI prompts to enhance their teaching practice. Whether you're designing assignments, creating assessments, or developing lesson plans, this collection offers practical solutions across multiple disciplines and teaching contexts.

## Features

- **Multilingual Support**: Access prompts in multiple languages for diverse educational environments
- **Subject-Specific Prompts**: Dedicated prompts for 14+ faculty areas and specialized topics
- **Cross-Functional Categories**: General teaching support, assessment, lesson planning, and more
- **Easy Navigation**: Web-based interface with search functionality for quick access
- **JSON-Based Structure**: Machine-readable format for easy integration and customization

## Getting Started

### For Users
1. Open `index.html` in a web browser
2. Use the search function to find relevant prompts
3. Browse by category or subject area
4. Copy and customize prompts for your specific needs

### For Developers
1. Review `prompts/index.json` to see available prompt categories
2. Each category has a corresponding JSON file with structured prompt data
3. Check `stiluri_pentru_web.md` for web styling guidelines
4. Integrate prompts into your applications using the JSON structure

## File Formats
